Concrete tech stack recommendations

Auth: JWT + refresh tokens, Passport strategies, bcrypt for PWs

Payments: Stripe (recommended), webhooks for transactional flow

File storage: AWS S3 or DigitalOcean Spaces; CDN (Cloudflare)

Search: Elasticsearch / OpenSearch

Queue: BullMQ + Redis

Frontend: Next.js + Tailwind CSS, shadcn/ui or Chakra UI for components

Observability: Sentry, Prometheus/Grafana, ELK for logs

Security & production hardening checklist (do these before public launch)

Use Helmet headers

Implement rate limiting & IP blocking for suspicious activity

Validate & sanitize all inputs (you already use class-validator)

Use refresh tokens and rotate tokens
